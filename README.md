This repository contains day-wise class notes, code examples, and projects created while learning full-stack front-end development. It starts from basic HTML tags and progressively builds up to CSS layouts, JavaScript DOM manipulation, jQuery, and complete interactive applications.

## Lessons Covered

### HTML
- Document structure, meta tags
- Headings, paragraphs, text formatting (`<strong>`, `<em>`, `<b>`, `<i>`, `<u>`)
- Lists (ordered, unordered, description)
- Links, images, tables, forms
- Semantic elements (`<code>`, `<pre>`, `<abbr>`, `<span>`, `<div>`)
- Audio, video, iframes
- HTML entities and special characters

### CSS
- Selectors, specificity, cascade
- Box model, margin, padding, borders
- Flexbox and Grid layouts
- Positioning (static, relative, absolute, fixed)
- Transforms (rotate, scale, translate)
- Transitions and animations
- Custom fonts with `@font-face`
- Media queries and responsive design

### JavaScript
- Variables, data types, operators
- Functions, conditionals, loops
- DOM selection and manipulation
- Event listeners and handling
- `setTimeout`, `setInterval`, `clearTimeout`, `clearInterval`
- Local Storage API
- jQuery (selectors, styling, DOM methods, events)

## Projects

| Project | Description | Concepts Used |
|---------|-------------|---------------|
| **Calculator** | A functional calculator with grid-based button layout | CSS Grid, DOM events, `eval()` |
| **Typing Game** | Speed typing game with countdown timer and scoring | `setInterval`, random selection, DOM updates |
| **To-Do List** | Task manager that persists data across sessions | localStorage, JSON, DOM creation |
| **Fan Animation** | Rotating fan blades using pure CSS | CSS transforms, `@keyframes` |
| **CSS Transitions Demo** | Animated div elements with transition effects | CSS transitions, JS class toggling |
| **Audio/Video + Bootstrap** | Media embedding with Bootstrap UI components | `<video>`, Bootstrap 5 grid, modals |
| **Class Timetable** | Weekly schedule table for group G23 | HTML tables, `rowspan`, `colspan` |

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/harshit391/HTML_Class.git
```

2. Open any `index.html` file in your browser to view that lesson or project

```bash
cd HTML_Class/Calculator
# Open index.html in your browser
```

No build tools or dependencies required. Everything runs directly in the browser.
